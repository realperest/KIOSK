# KIOSK

Raspberry Pi kiosk ve yerel termal yazıcı (Sıramatik). Pi yerleşimi: `/home/alper/apps/KIOSK/`.

- `ayarlamalar/` — Chromium tam ekran, otobaşlatma, `master_setup_raspberry.sh`
- `guncellemeler/` — Git çekme ve systemd ile yenileme betikleri
- `printer/` — ESC/POS HTTP aracı (`main.py`, port 9131)

Ayrıntı: `dosya_aciklamaları.md`, kurulum: `KURULUM.txt`.

Ana uygulama deposu: [SIRAMATIK](https://github.com/realperest/SIRAMATIK) — bu repodaki `KIOSK/` ile içerik paralel tutulur.
